### 3.2 锚文本规则

| 规则 | 示例 | 适用场景 |
|------|------|---------|
| **Exact Match (精确匹配)** | "ai loop video generator" → 指向 Pillar 1 | Support → Pillar 的首段 |
| **Partial Match (部分匹配)** | "create a seamless loop video" → 指向 HowTo 教程 | Pillar → Support 的正文引用 |
| **Brand + Keyword (品牌+关键词)** | "LoopCanvas AI loop video generator" → 指向 `/` | 所有文章底部 CTA |
| **CTA Anchor (行动号召)** | "Try it free" / "Create your loop now" → 指向 `/` | 正文中间悬浮 Bar |
| **Breadcrumb (面包屑)** | Home > Blog > Article | 所有文章顶部 |

### 3.3 禁止规则

- **禁止** Support 文章之间互相引用同层级文章（如 W3 不能指向 W5）
- **禁止** 死链闭环（所有文章最终必须能流出到 / 或 /spotify-canvas-maker 或 /pricing）
- **禁止** 过度优化锚文本（同一锚文本在同一文章内出现不超过 2 次）
